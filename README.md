# Financial-Sentiment-Analysis
AI/NLP project for financial sentiment analysis using classical ML and transformer-based models. The project compares Naive Bayes, Logistic Regression, SVM, DistilBERT, and FinBERT for financial text classification. 

The full project documentation was written in Obsidian using Markdown and exported to PDF in three languages: 
- Ukrainian 🇺🇦 (pages 1–24),
- Polish 🇵🇱 (24–48),
- and English 🇬🇧 (48–71).
