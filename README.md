# Financial-Sentiment-Analysis
AI/NLP project for financial sentiment analysis using classical ML and transformer-based models. The project compares Naive Bayes, Logistic Regression, SVM, DistilBERT, and FinBERT for financial text classification. 

The full project documentation was written in Obsidian using Markdown and exported to PDF in three languages: 
- Ukrainian 🇺🇦 (pages 1–24),
- Polish 🇵🇱 (24–48),
- and English 🇬🇧 (48–71).

Repository structure:

* `data.csv` — financial sentiment dataset
* `financial_sentiment_analysis.ipynb` — full NLP/ML workflow and model implementation
* `Sentiment analysis project 1.md` — complete project documentation written in Markdown
* `Financial-Sentiment-Analysis.pdf` — exported PDF version of the documentation (UA / PL / ENG)
* `README.md` — project overview and repository description
